# Design System — Sat Apps Marketplace

## 1. Brand Direction

Sat Apps Marketplace should feel:

- premium
- modern
- business-focused
- dark, elegant, glass-inspired
- clean and conversion-friendly

The UI should balance:
- strong product presentation
- polished readability
- clear actions
- reusable visual consistency across all screens

---

## 2. Core Brand Personality

### Visual keywords
- premium
- sleek
- trustworthy
- scalable
- product-first
- glassmorphism-inspired
- soft contrast
- minimal but rich

### Experience keywords
- easy to browse
- easy to compare
- easy to contact
- easy to convert

---

## 3. Color System

## Primary Colors

### Background
- `background`: `#0E0F12`
  - main app background

### Surface
- `surface`: `rgba(255,255,255,0.04)`
  - default soft card background

### Surface Elevated
- `surfaceElevated`: `rgba(255,255,255,0.06)`
  - stronger highlighted cards and preview blocks

### Primary Accent
- `primary`: `#B87A56`
  - main brand accent
  - used for buttons, badges, highlights, prices

### Primary Soft
- `primarySoft`: `#D39A74`
  - gradient start for CTA buttons

---

## Text Colors

### Main text
- `textPrimary`: `#F6F3EF`

### Secondary text
- `textSecondary`: `rgba(255,255,255,0.72)`

### Muted text
- `textMuted`: `rgba(255,255,255,0.48)`

### Dark text
- `textDark`: `#1A1A1A`
  - used on primary gradient buttons

---

## Border Colors

### Standard border
- `border`: `rgba(255,255,255,0.08)`

### Highlight border
- `borderHighlight`: `rgba(184,122,86,0.40)`

---

## Support Colors

### White soft overlay
- `whiteSoft`: `rgba(255,255,255,0.05)`

### Black overlay
- `blackOverlay`: `rgba(10,12,16,0.30)`

### Gold highlight layer
- `accentOverlay`: `rgba(184,122,86,0.12)`
